ubuntu用shadowsocks科学上网完全教程

发表于 2017-07-11 | 分类于 工具们 |

声明：科学文明上网，本文只分享技术，请自行承担后果

前言：

之前一直使用Windows做学习的，这段时间在学git和python，于是在笔记本上划了50G空间又装了个Ubuntu16.04 LTS，之前之前在别的机器和虚拟机上尝试过安装shadowsocks QT5的客户端，但是尝试过几次都没有成功番羽墙成功。这次用了好久来找博客，终于成功。以下是我的步骤：

  1. shadowsocks是目前比较稳定的代理方式，首先你要有一台代理服务器（vps），或者买一个shadowsocks账号。这个步骤在网上一搜一大把

  2. 安装Shadowsock客户端，我用的是shadowsocks-qt5，因为是图形界面版，便于管理

    • 在终端(ctrl+alt+t) 中依次输入（直接粘贴复制）三个步骤（分别是：添加源->更新源->安装）：
    • sudo add-apt-repository ppa:hzwhuang/ss-qt5
    • sudo apt-get update
    • sudo apt-get install shadowsocks-qt5

  3. 设置代理服务

    • 打开软件shadowsocks-qt5
    • 连接->添加->手动 添加配置信息
    • 填写服务器地址，服务器端口，密钥，加密方式，其他的默认即可，配置名称随便写，点击OK
    • 点击连接即可开启服务，如果报错请检查上一步填写的信息是否有误，或者你的代理服务器是否正常服务

  4. 至此，shadowsocks已经配置完成，接下来设置全局代理，以下两步是准备工具：

    1. 安装pip：(upgrade前的横线是两个短横线-，直接复制会出错)

      *   <div>sudo apt-get install python-pip python-dev build-essential</div>
      sudo pip install –upgrade pip

      sudo pip install –upgrade virtualenv

      1. 安装GenPAC：

        • sudo pip install genpac

          sudo pip install –upgrade genpac

      2. 现在要下载一个文件，首先cd命令进入到你想存放文件的目录，比如:cd /home/yanhui

    2. 执行下面命令：

      *   sudo genpac --proxy="SOCKS5 127.0.0.1:1080" --gfwlist-proxy="SOCKS5 127.0.0.1:1080" -o autoproxy.pac --gfwlist-url="https://raw.githubusercontent.com/gfwlist/gfwlist/master/gfwlist.txt"
      1. 现在在/home/yanhui的目录下生成了一个autoproxy.pac的文件

    3. 设置全局代理：

      1.  系统设置-&gt;网络-&gt;网络代理
      1. 方法选择“自动”
      2. 配置url填写：file:///home/yanhui/autoproxy.pac

      3. 点击应用到整个系统

      4. 用google来测试是否成功
