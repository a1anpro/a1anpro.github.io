mysql数据库修改密码和忘记密码如何登陆

发表于 2017-11-02 | 分类于 工具们 |

在做课程设计的时候，用到了数据库，在修改mysql登陆密码后，忘了刷新权限，导致了登陆不上去。本文介绍如何修改mysql数据库密码，以及忘记密码如何登陆。

1：如何修改mysql数据库密码：

登陆mysql : mysql -u root -p

选用mysql数据库：use mysql;

更改密码：update user set password=password(‘123456’) where user=’root’ ;

更改密码之后，需要刷新权限，否则登陆不了：flush privileges;

 

2：如何忘记密码，如何登陆mysql：

首先，需要关闭MySQL Server服务。在“运行”窗口，输入“services.msc”，进入“服务”窗口。

在服务窗口，可以找到“MySQL”，选定，将其“停止”即可。这样，MySQL Server服务就停止了。

找到mysql client的安装路径，笔者的安装路径是：C:\Program Files (x86)\MySQL\MySQL Server 5.5

知道安装路径是为了访问.ini文件，该文件记录了MySQL的配置信息。

需要注意的是，不同版本的.ini文件的名称不同，在MySQL 5.5时，它是my.ini。而在MySQL 5.6则是my-default.ini

接着，打开命令行窗口，输入如下的命令：mysqld –defaults-file=”C:\Program Files (x86)\MySQL\MySQL Server 5.5\my.ini” –console –skip-grant-tables

需要说明的是，此时该窗口不可再写入。这也是检验命令是否正确执行的一个标准。

另外打开一个命令行窗口，输入命令：mysql -u root -p，回车即可进入mysql命令行界面。

如此，就已经可以免密登陆mysql了，再用步骤1中的修改密码的方法修改密码，并记住：一定要刷新权限。
