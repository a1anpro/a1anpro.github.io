anaconda环境命令笔记

发表于 2017-10-19 | 分类于 python学习 ， 工具们 |

#环境：ubuntu16.04
conda是anaconda下用于包管理和环境管理的工具，功能上类似pip和virtualenv的组合。安装成功后conda会默认加入到环境变量中，因此可以直接在命令行窗口运行命令conda

#创建python3.6版本创建一个名字为python36的环境
conda create –name python36 python=3.6

#激活环境
source activate python36

#查看python版本
python -V

#退出该环境
deactivate python36

#删除该环境
conda remove -n python –all

#查看所有安装的环境
conda info -e

conda的包管理和pip一样：

#安装matplotlib
conda install matplotlib

#查看所有安装了的包
conda list

#包更新
conda update matplotlib

#删除包
conda remove matplotlib

 

在conda中，anything is a package,conda本身也可以被看做是一个包，python环境也是一个包，anaconda也可以看做是一个包。因此除了普通的第三方包支持更新之外，这三个包也支持更新：

#更新conda本身
conda update conda

#更新anaconda应用
conda update anaconda

#更新python
conda python update
